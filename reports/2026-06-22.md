### 产品方向

- **TokenOps** — 面向 AI 团队的 FinOps 仪表盘，实时追踪 token 消耗并按功能/团队归因，防止 Agent 循环导致费用失控
  - 目标用户：AI 产品工程负责人、FinOps 与财务团队、多 Agent 应用初创公司
- **AI unit economics** — 面向 CFO 与工程负责人的 AI 单位经济学平台，将推理费用按产品/功能拆解并近实时告警异常支出
  - 目标用户：SaaS 公司 CFO、工程 VP、AI 产品财务分析师
- **physical-AI serving** — 面向机器人与边缘设备的低延迟物理 AI 推理服务框架，支持小批量端侧部署与执行状态检查点恢复
  - 目标用户：机器人系统工程师、边缘 AI 部署团队、物理世界 Agent 开发者
- **memory governance** — 面向 Agent 开发者的可治理分层记忆系统，解决长期记忆缺乏过期策略与跨会话上下文污染问题
  - 目标用户：AI Agent 开发者、企业级 LLM 应用团队、多会话客服 Agent 运营方
- **Surya OCR 2** — 面向开发者的生产级 OCR 工具，提供全页处理实时进度、稳定依赖兼容与批处理一致性保障
  - 目标用户：文档处理开发者、RAG 管道工程师、多语言 OCR 集成团队
- **Handit.ai** — 面向运营团队的 Agent 任务流水线平台，提供可验证、可追踪的多步 Agent 编排与人工交接机制
  - 目标用户：运营自动化负责人、AI Agent 产品团队、需要审计 Agent 输出的合规团队
- **Agent 37 Cloud** — 面向知识工作者的主动协作云端 AI 同事，消除碎片化被动工具间的频繁 tab 切换与上下文丢失
  - 目标用户：远程协作团队、知识工作者、小型创业团队运营负责人
- **Atomic Mail Agentic** — 面向 B2B 团队的行业专用邮件 Agent 平台，稳健处理非正式语言并提供可定制的收件箱自动化模板
  - 目标用户：B2B 销售团队、客户成功经理、需要邮件自动化的中小企业
